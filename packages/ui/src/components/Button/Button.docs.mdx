# Button

## Status

{buttonMeta.status} is the current alignment state for the component. The implementation follows SwiftUI's `Button` concept while keeping the web-native `onClick` interaction model.

## SwiftUI Alignment

Button maps directly to SwiftUI's `Button` control: a label or icon triggers an action when activated. The React version keeps the same mental model while exposing standard HTML button behavior.
